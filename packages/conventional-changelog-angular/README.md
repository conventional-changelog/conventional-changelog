**Issues with the convention itself should be reported on the Angular issue tracker.**

## Install

```bash
# pnpm
pnpm add -D conventional-changelog-angular
# yarn
yarn add -D conventional-changelog-angular
# npm
npm i -D conventional-changelog-angular
```

## Usage

Use with [conventional-changelog](https://conventional-changelog.js.org/conventional-changelog/) by passing the preset name with `-p`:

```sh
conventional-changelog -p angular
```

## Documentation

For the commit convention details and preset options, visit the [documentation website](https://conventional-changelog.js.org/presets/angular/).
